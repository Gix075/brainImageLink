brainImageLink

is a nice Joomla Module able to show image carousels on your site, thinked for your customers!

  • Simple to use
  • Fluid/Responsive
  • Joomla 3 ready
  • Thinked to offer a user friendly module for your customers

Usage

Download and install the zip pack

In order to use this module, just download it and install the zip pack from the joomla extensions manager

Go to the module configuration

Go to the modules administration then add and configure any modules you need. Module configuration is really simple and intuitive. There are two main tabs to use: Module and Style Configuration

Module Tab

This is the main tab. Here you can configure the mandatory module fields. Here you can choose the link kind (internal or external), insert the link url (or choose a menu item), select and/or update an image, set the alt and title attributes, activate or not the joomla tooltip, activate or deactivate the rel='nofollow' tag and much more.

Style Config Tab

This module offers you some different style: default style (that consits in a no-style module) and other styles, such as Overlay style. For each non default style you can choose a color preset or make your own by the colors configuration.

Responsive Module

This module, as joomla 3 requires, is complitly responsive for all devices.

Highly Customizable

This module is complitly customizable from admin panel. You don't need to know html or css or javascript to use and customize brainImageLink

Custom Icons

This module provide an additional style with an overlay effect.
On overlay you can activate some text and one custom icon that you can choose from:
Icomoon , Bootstrap, Font Awesom
